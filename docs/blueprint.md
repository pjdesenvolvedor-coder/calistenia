# **App Name**: Quiz de Produto Móvel

## Core Features:

- Criação de Quiz: Permitir que administradores criem quizzes com múltiplas perguntas e respostas.
- Visualização Mobile: Assegurar que o quiz seja totalmente responsivo e otimizado para visualização em dispositivos móveis.
- Apresentação de Perguntas: Mostrar as perguntas e respostas de forma clara e organizada.
- Registro de Respostas: Capturar e registrar as respostas do usuário em cada pergunta.
- Cálculo de Resultados: Calcular a pontuação do usuário com base nas respostas corretas e apresentar um resultado imediato.
- Recomendação de Produto: Com base nas respostas do quiz, o sistema recomendará o produto mais adequado para o usuário, utilizando um tool para analisar as respostas.
- Compartilhamento Social: Permitir que os usuários compartilhem seus resultados nas redes sociais.

## Style Guidelines:

- Cor primária: Azul vibrante (#29ABE2) para transmitir confiança e modernidade.
- Cor de fundo: Azul claro (#F0F8FF) para suavizar o visual e melhorar a legibilidade.
- Cor de destaque: Laranja (#FF9933) para botões de ação e elementos importantes.
- Fonte para títulos: 'Poppins', sans-serif, para um visual moderno e limpo.
- Fonte para corpo do texto: 'PT Sans', sans-serif, para garantir a legibilidade em dispositivos móveis.
- Ícones minimalistas e claros, que representem cada seção do quiz.
- Design responsivo com foco na experiência mobile, utilizando um layout de uma coluna para facilitar a navegação.
- Transições suaves entre as perguntas para manter o engajamento do usuário.
